XTC101 - Introduction to Testing and Test Cloud

Overview

This class introduces the student to how to test applications using NUnit, Xamarin.UITest and Calabash and then have those tests upload to Test Cloud to see how those tests would perform on a wider range of devices.

Estimated Time: 2 Hours

Objectives

  1. Introduction to Testing
    Worksheet
    In this exercise, you will learn the how to create tests, how to understand the unique aspects of testing mobile apps.
  2. Unit Testing
    Worksheet Exercise  
    In this exercise, you will understand how to create unit tests with NUnit, how to name and structure them as well as best practices.
  3. Xamarin.UITest
    Worksheet
    In this exercise, you will learn how to test the user interface of your apps with Xamarin.UITest. You will learn how to run your app on the simulator or device, how to use the REPL tool to find views on your User Interface.
  4. Calabash
    Worksheet
    In this exercise, you will be introduced to Cucumber, a framework for defining specs in a business friendly language called Gherkin and then have those tests execute either using built in steps or using Ruby code.
  5. Test Cloud
    Worksheet
    In this exercise we will use the tests that we have executed locally and then upload them to Test Cloud to see how they would perform on a wider range of devices.

Setup

To complete the coding exercise for this class, you will need either an Android or an iOS development setup.

  • iOS
    You will need a computer with Mac OS X, Apple's XCode development environment and Xamarin 3.0 installed to be used as the Build Machine. You can then either use this same machine with Xamarin Studio, or you can use a Windows machine with the Xamarin tools and Visual Studio to do the code editing.
  • Android
    You will need either a Mac OS X computer with Xamarin tools and the Android SDK, or a Windows computer with Visual Studio, Xamarin tools and the Android SDK. In addition, if you do not have an Android device, you will need to setup the Android emulator, Genymotion virtual machine, or use the built-in Xamarin Android Player beta.
  • Calabash
    You will need to install the Calabash tools on your Mac on Windows machine to allow for local testing of your iOS or Android apps. Instructions for installation can be found here

If you need help setting up your environment, try going through the online setup instructions. In addition, we have documented all the setup requirements and have walk-through videos on the Xamarin University website.
