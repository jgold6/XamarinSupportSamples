Introduction to Android with Xamarin Studio

Exercise 1: Create a Xamarin.Android application in Xamarin Studio

Duration

15 minutes

Lab goals

The goal of this lab is to walk through the creation of a Xamarin.Android application using Xamarin Studio. We will examine the solution structure, discuss each part of the source code, then build and run the app.

This is a code-along lab, which should be done in tandem with an online instructor for Xamarin University. If you are working through this class on your own, you can view the steps in the recorded video.
