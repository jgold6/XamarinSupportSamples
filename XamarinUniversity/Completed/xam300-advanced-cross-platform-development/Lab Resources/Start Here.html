Advanced Cross Platform Mobile Development

Overview

This session builds on the introduction class and goes deeper into architectural patterns used to abstract our dependencies which can help with cross platform development as well as unit testing your shared code.

Duration

3 hours

Objectives

  1. Working with Dependencies in a Loosely-Coupled fashion

Lecture

XAM300 - Advanced Cross Platform Mobile Development.

Exercises

  1. Using the Factory Pattern
    Worksheet   Exercise
    Create a dependency through the Factory pattern.
  2. Build a Service Locator
    Worksheet   Exercise
    Add an abstraction and Service Locator into our project to find and use a platform-specific feature.
  3. Using Dependency Injection
    Worksheet   Exercise
    Switch to a DI/IoC container and inject our dependency into our code.

Setup

This is a cross-platform class which can use either iOS or Android. To complete the coding exercise for this class, you will need either a Windows or OS X setup.

  • Xamarin Studio + Mac OS X
    You will need a computer with Mac OS X, Apple's XCode development environment, the Android SDK and Xamarin 3.0 installed. You will then be able to execute the iOS and Android projects.
  • Visual Studio + Windows
    You will need a Windows computer with Visual Studio 2013 (Update 2) with Xamarin 3.0 installed to run the Android project. In addition, to execute the iOS project, you will need a Mac OS X build machine.

If you need help setting up your environment, try going through the online setup instructions. In addition, we have documented all the setup requirements and have walk-through videos on the Xamarin University website.
