Portable Class Libraries
Portable Class Libraries (PCLs) are a way of sharing binary assemblies with different .NET based system.

Dependency Injection
One downside to using Portable Class Libraries it they will limit the APIs you can use. If you must utilize a platform API which is not available, you can turn to Dependency Injection to provide access to the unavilable API.
.NET Resources
Sharing resources in Portable Class Libraries requires a common way of accessing them across all target platforms.
