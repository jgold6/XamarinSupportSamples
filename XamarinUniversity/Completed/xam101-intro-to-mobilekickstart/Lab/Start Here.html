Introduction to Mobile Development / Kickstarter

Overview

This class is a quick introduction to Xamarin and Mobile Development for iOS and Android. It covers the Mobile Application Development Lifecycle using Xamarin tools and is the starting point for Xamarin University training.

Duration

2 Hours

Lecture

XAM101 - Introduction to Mobile Development.

Objectives

  1. Hands on with Xamarin
  2. How Xamarin works
  3. Xamarin University

Exercises

  1. Hands-On with Xamarin
    Exercise
    In this exercise, we will build and run an existing application that targets both iOS and Android. This will allow you to test your development environment, and along the way, get a cool new T-Shirt!
  2. How Xamarin Works
    Worksheet
    We will discuss the tools and build process for Xamarin applications.
  3. Xamarin University
    Finally, we will talk about some of the benefits of Xamarin University, the certification process and expectations going forward.

Setup

To complete the coding exercise for this class, you will need either an Android or an iOS development setup.

  • iOS
    You will need a computer with Mac OS X, Apple's XCode development environment and Xamarin 3.0 installed to be used as the Build Machine. You can then either use this same machine with Xamarin Studio, or you can use a Windows machine with the Xamarin tools and Visual Studio to do the code editing.
  • Android
    You will need either a Mac OS X computer with Xamarin tools and the Android SDK, or a Windows computer with Visual Studio, Xamarin tools and the Android SDK. In addition, if you do not have an Android device, you will need to setup an Android emulator to run the lab exercise.

If you need help setting up your environment, try going through the online setup instructions. In addition, we have documented all the setup requirements and have walk-through videos on the Xamarin University website.
