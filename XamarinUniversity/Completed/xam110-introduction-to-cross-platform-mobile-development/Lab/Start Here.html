Intro to Cross-Platform Mobile Development

Overview

This class is meant as a follow up to the Intro to iOS and Android classes, this course introduces how to create solutions that have multiple platform targeted applications and how to maximize code sharing between them.

Duration

3 Hours

Objectives

  1. Working with Shared Components
  2. Using Shared Projects
  3. Using Portable Class Libraries

Lecture

XAM110 - Intro to Cross Platform Mobile Development.

Exercises

  1. Nuget and the Xamarin Component Store
    Worksheet  
    Quick demonstration of sharable components available from Nuget and the Xamarin Component Store.
  2. Working with Shared Projects
    Worksheet   Exercise
    One way to share code across platforms is to compile the same source files into each project. In this exercise, we will examine sharing code at the source-level with the new Shared Project feature in Visual Studio 2013 Update 2 and Xamarin Studio 5.
  3. Working with Portable Class Libraries
    Worksheet   Exercise
    A completely different style of code sharing is to share assemblies between projects - in this section, we will look at Portable Class Libraries and how to use them effectively in your projects to maximize the amount of code we can share.

Setup

This is a cross-platform class which has projects for iOS, Android, and Windows Phone 8. To complete the coding exercise for this class, you will need either a Windows or OS X setup, or both to use all three platforms.

  • Xamarin Studio + Mac OS X
    You will need a computer with Mac OS X, Apple's XCode development environment, the Android SDK and Xamarin 3.0 installed. You will then be able to execute the iOS and Android projects.
  • Visual Studio + Windows
    You will need a Windows computer with Visual Studio 2013 (Update 2) with the Windows Phone 8 SDK and Xamarin 3.0 installed. In addition, to execute the iOS project, you will need a Mac OS X build machine.

If you need help setting up your environment, try going through the online setup instructions. In addition, we have documented all the setup requirements and have walk-through videos on the Xamarin University website.
