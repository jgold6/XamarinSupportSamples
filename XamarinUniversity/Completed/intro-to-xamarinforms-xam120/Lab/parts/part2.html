Introduction to Xamarin.Forms

Exercise 2: Creating our first Xamarin.Forms application

Duration

15 minutes

Goals

The primary goal is to create a new Xamarin.Forms application using the provided project templates in Visual Studio or Xamarin Studio and then going through the created project to understand all the parts.

This is a group exercise and is intended to be done together with a Xamarin University instructor. You can also watch the recorded session to follow along with the exercise.
