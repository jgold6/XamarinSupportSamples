ListViews and Adapters in Android

Overview

This session introduces Android ListViews and Adapters. We start with the core tasks involved in using an Adapter to populate a ListView. Then we show a few patterns to boost efficiency and some techniques to help the user navigate a large data set. We finish by demonstrating how to respond when the user clicks on an item in the list.

Duration

2.5 hours

Objectives

  1. Populate a ListView using an ArrayAdapter
  2. Handle list-item click events
  3. Implement a custom adapter
  4. Use layout recycling and the view-holder pattern
  5. Enable fast scrolling and code a section indexer

Lecture

AND110 - ListViews and Adapters in Android

Exercises

  1. Populate a ListView using an ArrayAdapter
    Worksheet   Exercise
    In this first lab, you will create a new project, integrate the provided sample data, and use an ArrayAdapter to display the data in a ListView.
  2. Handle list-item click events
    Worksheet   Exercise
    Here you will subscribe to the ListView.ItemClick event. When the user clicks on an item in the list, you will display an alert dialog that shows which item was clicked.
  3. Implement a custom adapter
    Worksheet   Exercise
    Here you will implement a custom adapter that will build custom rows for a ListView.
  4. Use layout recycling and the view-holder pattern
    Worksheet   Exercise
    This section shows you two common techniques used with ListView and custom adapters to increase efficiency: layout recycling and view holders.
  5. Enable fast scrolling and code a section indexer
    Worksheet   Exercise
    In this part, you will enable two ListView features to help the user navigate through a large data set: fast scrolling and a section indexer.
