Introduction to Xamarin.Forms

Overview

This class is designed to introduce you to the features and architecture of Xamarin.Forms, a cross-platform UI toolkit for iOS, Android and Windows Phone 8. We will be covering basic app creation, the layout model, using platform-specific features and basic navigation.

Duration

3 hours

Lecture

XAM120 - Introduction to Xamarin.Forms.

Objectives

  1. Cross Platform UI Strategies
  2. Xamarin.Forms App Structure
  3. Pages, Controls and Layout
  4. Using Platform Specific Features

Exercises

  1. Cross Platform UI Strategies
    Worksheet  
    In this exercise, we will contrast the platform-specific UI Xamarin model you have used in the past, with the new Xamarin.Forms model and introduce the features of the platform.
  2. Creating our first Xamarin.Forms application
    Worksheet   Exercise
    In this exercise, we will create basic Xamarin.Forms application together using the built-in project templates and go through the solution structure.
  3. Creating Xamarin.Forms Phoneword
    Worksheet   Exercise
    In this exercise, you will create a new Xamarin.Forms application (Phoneword) to enter an alphanumeric phone number which is then translated to a standard numeric telephone number.
  4. Adding support for dialing the phone
    Worksheet   Exercise
    In this final exercise, you will extend your Phoneword application by adding support to dial the phone and display an alert view cross-platform.

Setup

To complete the coding exercise for this class, you will need an Android, Windows Phone 8, or an iOS development setup, ideally at least two of these.

  • Android
    You will need either a Mac OS X computer with Xamarin tools and the Android SDK, or a Windows computer with Visual Studio 2013 Update 2, the Xamarin 3.0 tools and the Android SDK. In addition, if you do not have an Android device, you will need to setup the Android emulator, Genymotion virtual machine, or use the built-in Xamarin Android Player beta.
  • Windows Phone
    You will need a Windows computer with Visual Studio 2013 Update 2 and Xamarin 3.0 installed. In addition, you will need the Windows Phone 8.0 or 8.1 SDK installed.
  • iOS
    You will need a computer with Mac OS X, Apple's XCode development environment and Xamarin 3.0 installed to be used as the Build Machine. You can then either use this same machine with Xamarin Studio, or you can use a Windows machine with the Xamarin 3.0 Tools and Visual Studio 2013 Update 2 to do the code editing.

If you need help setting up your environment, try going through the online setup instructions. In addition, we have documented all the setup requirements and have walk-through videos on the Xamarin University website.
