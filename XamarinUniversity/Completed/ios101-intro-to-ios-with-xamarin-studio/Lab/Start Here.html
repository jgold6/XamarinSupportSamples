Introduction to iOS with Xamarin Studio

Overview

Introduces iOS development on the Mac using Xamarin Studio. This session covers basic app creation, Apple's Model-View-Controller pattern, the iOS Storyboard Designer, deployment, debugging, and app fundamentals.

Duration

2.5 Hours

Lecture

iOS101 - Introduction to iOS with Xamarin Studio.

Objectives

  1. Create a Xamarin.iOS Application
  2. Use the Xamarin iOS Designer
  3. Implement behavior in code behind
  4. Navigating to a second screen
  5. Adding app icons and a splash screen

Exercises

  1. Creating our first iOS application with Xamarin Studio
    Worksheet   Exercise
    In this exercise, we will build a simple iOS application and go through all the parts of the project so you understand how it is constructed, built and executed.
  2. Designing the Phoneword Application
    Worksheet   Exercise
    In this exercise, you will create a new iOS application and design the user interface using the Storyboard Designer.
  3. Adding Behavior to Phoneword
    Worksheet   Exercise
    In this exercise, you will add behavior into the Phoneword application you designed earlier.
  4. Adding a second screen to Phoneword
    Worksheet   Exercise
    In this exercise, you will add a second screen to display Call History into your Phoneword application.
  5. Adding Final Touches
    Worksheet   Exercise
    In this final exercise, we will add the final touches to our Phoneword application and prepare it for deployment by adding icons, a splash screen and setting up the application metadata.

Setup

You will need a computer with Mac OS X, Apple's XCode development environment and Xamarin 3.0 installed. If you need help setting up your environment, try going through the online setup instructions. In addition, we have documented all the setup requirements and have walk-through videos on the Xamarin University website.
