Introduction to Android with Xamarin Studio

Overview

This session introduces Android development using Xamarin Studio. It covers the basics of creating a new app, adding controls to a UI, handling events, communicating between multiple screens, utilizing Android Resources, deployment, and execution.

Duration

2.5 hours

Objectives

  1. Create a Xamarin.Android application in Xamarin Studio
  2. Use the Xamarin.Android Designer to add controls to a UI
  3. Implement app behavior in C# code-behind
  4. Code two screens and navigate between them
  5. Use Resources to incorporate custom labels and icons

Lecture

AND101 - Introduction to Android with Xamarin Studio

Exercises

  1. Create a Xamarin.Android application in Xamarin Studio
    Worksheet   Exercise
    Build a simple Android application and explore each part of the project so you understand how it is organized, built, and executed.
  2. Use the Xamarin.Android Designer to add controls to a UI
    Worksheet   Exercise

  3. Implement app behavior in C# code-behind
    Worksheet   Exercise
    Add behavior to the Phoneword application you designed earlier.
  4. Code two screens and navigate between them

  5. Use Resources to incorporate custom labels and icons
    Worksheet   Exercise

Setup

You will need either a Mac or Windows computer with Xamarin installed, including the Android SDK and Xamarin Studio. If you need help setting up your environment, try going through the online setup instructions. In addition, we have documented all the setup requirements and have walk-through videos on the Xamarin University website.
